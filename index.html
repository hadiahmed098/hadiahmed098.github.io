USB Rx/Tx Peripheral

A USB 2.0 Full Speed Peripheral connected through an AHB-Lite bus written in Verilog. Implements FSMs, bit stuffing for timing synchronization, and CRC error detection. The project also contains SystemVerilog component and design level test benches to verify the design. After synthesis using Mentor Graphics Questa, I further verfied the design and checked STA reports for positive slack.

PongAI

A General Adverserial Network written in Python to play Pong against a human or computer player. The GAN has three layers - the screen's RGB data is the input layer and the output layer has two nodes: move up or move down. The program was written using OpenCV2 and NumPy and trained on a Pong clone using PyGame.

Steganographic Encoder

A Python command line tool to encode the data from a .txt file into the least significant bits of each pixel in a .png image file. The tool also can decode the data stored in the lest significant bits of each pixel of a .png image and output the results to a .txt file. Part of the encoding and decoding process allows the user to decide how many bits of each pixel are used to store the data.

Triangle Clock

A geometric clock written in Java and Java Swing. The hands of an analog clock are replaced with the verticies of a triangle. As time progresses, the verticies move around the clock and are interconnected, forming a triangle. Each vertex is also color-coded to help read the time from the triangle position. Finally, the clock includes a color-coded label.

github.com/hadiahmed098/TriangleClock
SimpleCipher

A text cipher written in Java that uses the Java Swing libraries. It manipulated text encodings to transform Latin characters into random UTF-16 Unicode characters. Text is encoded by converting each character into morse code, assigning each character a psuedorandom number, and converting the numbers into a Unicode string.
